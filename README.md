## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/kimsuhan/mysql-table-mybatis-create.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Create your Enviroment in `.env`
   ```env
   DB_HOST={host}
   DB_PORT={port}
   DB_USER={user}
   DB_PASSWORD={password}
   ```
